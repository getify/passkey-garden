Try customizing these buttons...

To pick a passkey register/login button that works best, try out different icons, color schemes, sizes, and supporting text.

You can also upload -- locally in your browser -- other .svg icons to evaluate their legibility/usability under these various conditions. Maybe your design will end up being the icon we all pick to standardize on!

To upload an icon, make sure it's a single-color SVG, transparent background, and square aspect ratio (any size is fine). For best results, also make sure it's solid black color (#000), and that it has a proper width, height, and viewbox attributes defined. Check out an example icon file for compatible file format/contents to upload.
